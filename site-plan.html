TrailFinder Site Plan

Benito Juarez

Site Name

TrailFinder

I chose this name because it is short, action-oriented, and clearly describes the service. It immediately tells the user that this is a utility for finding hiking locations.

Optional domain: trail-finder-app.com

Site Purpose

The site provides a comprehensive directory for outdoor enthusiasts to explore National Parks and hiking trails. It allows users to filter trails by difficulty, view detailed statistics (elevation, rating), and save their favorite locations to a personal "Bucket List" stored in their browser.

Scenarios

  • Scenario 1: "I am a beginner hiker looking for a scenic trail that isn't too difficult. How can I find trails rated 'Easy' near a specific park?"
  • Scenario 2: "I am planning a trip for next summer. I found a park I like on the site; is there a way to save it so I don't lose it?"

Color Scheme

The color palette is inspired by nature maps and forestry. I am using a dark green for authority and structure, a soft beige for the background (to reduce eye strain), and a terracotta orange for call-to-action buttons.

Typography

Heading Font: Montserrat

I chose Montserrat for headings because it is a geometric sans-serif that feels modern and sturdy, similar to signage found in National Parks.

Body Font: Open Sans

I chose Open Sans for body text because it is highly legible on screens and has a friendly, neutral tone that pairs well with the bold headers.
